常用Git命令

mkdir 项目名字（新建）

vi 文件名（修改文件）

cat 文件名字（查看文件）

rm -rf 文件名（删除文件）

git clone  git@git.coding.net:clm1010/js-git-test.git（克隆项目）

git add . （添加要上传的文件 .代表全部，也可以是 文件名）

git diff（查看修改了哪些）

git status （查看状态）

git commit -m "注释" （提交注释）

git push origin 分支名字 （添加文件到分支）

git pull origin 分支名字（更新分支）

git branch（查看分支）

git checkout -b  分支名字（新建分支 -b就是branch）

git merge 要合并的分支名字（合并）
